# champ V2 Backend

This is the backend for **CHAMP-MD V2**, a WhatsApp AI bot project by 〔𝗢𝗚 𝗖𝗛𝗔𝗠𝗣〕√.  

It handles AI processing, session management, media handling, and more.  

---

## 🚀 Deploy to Heroku

You can deploy this backend instantly to Heroku using the button below:

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/Daniel432-gift/creepy-v2-backend)

> Click the button and it will open Heroku with this repository already selected for deployment.

---

## 📦 Features

- WhatsApp session management
- AI conversation handling
- Media (images, audio, video) support
- Voice-to-text and text-to-voice integration
- Secure environment variable management

---

## ⚡ Requirements

- Node.js 18+
- MongoDB (Atlas or local)
- Heroku account (for deployment)
